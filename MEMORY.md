## Суть проекта

**LeanPro** — коммерческий образовательный сайт по бережливому производству (Lean Manufacturing).
Целевая аудитория: руководители цехов, инженеры, специалисты по улучшению процессов на российских предприятиях.

Сайт продаёт онлайн-курсы через форму заявки + интеграцию с **ЮKассой** (YooKassa).
Деплой: **GitHub Pages** → репозиторий `https://github.com/dmitrybar140984-cmyk/lean-site`
Планируемый домен: **leanpro.ru**

---

## Стек технологий

- Чистый HTML5 + CSS3 + Vanilla JS (без фреймворков, без npm, без сборщиков)
- Шрифт: **Inter** через **Bunny Fonts** (`fonts.bunny.net`) — НЕ Google Fonts (заблокирован/медленный в РФ)
- Никаких внешних CDN для JS (всё встроено в файлы)
- Никаких аналитических сервисов Google/Facebook (нет в коде)

## Дизайн-система

### CSS-переменные (из styles.css)
```css
--primary:       #1a6b3c   /* тёмно-зелёный, основной */
--primary-dark:  #134d2c   /* hover состояния */
--primary-light: #e8f5ee   /* фоны, бейджи */
--accent:        #f5a623   /* оранжевый, акцент */
--text:          #1a1a2e   /* основной текст */
--text-muted:    #6b7280   /* вторичный текст */
--bg:            #ffffff
--bg-alt:        #f8faf9   /* секции с чуть серым фоном */
--border:        #e5e7eb
--radius:        12px
--shadow:        0 4px 24px rgba(0,0,0,0.08)
--shadow-lg:     0 12px 48px rgba(0,0,0,0.12)
```

### Типографика
- Шрифт: Inter (400, 500, 600, 700, 800)
- Подключение: `https://fonts.bunny.net/css?family=inter:400,500,600,700,800&display=swap`
- Базовый размер: 16px

### Кнопки
- `.btn--primary` — зелёный фон, белый текст
- `.btn--ghost` — прозрачный с зелёной рамкой
- `.btn--white` — белый фон, зелёный текст (на тёмных секциях)
- Модификаторы: `.btn--sm`, `.btn--lg`, `.btn--full`

## Юридические страницы (для ЮKassы)

Все три страницы созданы и подключены в футере и формах.

### Что нужно заполнить (плейсхолдеры)
В `privacy.html`, `offer.html`, `refund.html` замените:
- `[УКАЖИТЕ ИНН]` — ИНН организации
- `[УКАЖИТЕ ОГРН]` — ОГРН
- `[УКАЖИТЕ КПП]` — КПП
- `[УКАЖИТЕ ЮРИДИЧЕСКИЙ АДРЕС]` — полный юр. адрес
- `[УКАЖИТЕ НОМЕР ЛИЦЕНЗИИ]` — номер лицензии на образовательную деятельность
- `[УКАЖИТЕ РАСЧЁТНЫЙ СЧЁТ]`, `[УКАЖИТЕ БАНК]`, `[УКАЖИТЕ БИК]`, `[УКАЖИТЕ КОРР. СЧЁТ]` — банковские реквизиты

Также в `index.html` в `footer__legal` есть те же плейсхолдеры.

### Согласие в формах
Обе формы (контактная + модальная покупка) содержат:
```html
<a href="privacy.html">политикой обработки персональных данных</a> и <a href="offer.html">договором оферты</a>
```

## Правила для ИИ, работающих над проектом

1. **Никаких Google-зависимостей** — шрифты только через `fonts.bunny.net`, без Google Analytics, без gtag
2. **Никаких npm/сборщиков** — всё чистый HTML/CSS/JS, встраивается напрямую в файлы
3. **Русский язык** — весь контент сайта на русском, включая комментарии в коде
4. **Стили новых страниц** — переиспользовать переменные из `styles.css`; для standalone-страниц (game, vsm, ladm, lean-flow, quiz) стили инлайн в `<style>` внутри файла
5. **Юридические страницы** — шаблон уже есть в `privacy.html`, копируй структуру (`legal-page`, `legal-box`, `back`, навбар, футер)
6. **После каждого изменения** — обновляй раздел «История изменений» в этом файле и коммить его вместе с остальными файлами
7. **Цена курсов** — не менять без явного указания пользователя
8. **Интерактивные инструменты** — каждый открывается из карточки курса через `course-card__game-btn`, ведёт на отдельный HTML-файл
9. **Закон о рекламе РФ** — НЕ использовать утверждения «№1», «лучший», «единственный» и аналоги без документального подтверждения (ФАС)
10. **Наименования компаний** — НЕ упоминать реальные торговые марки и юрлица в разделе партнёров без письменного согласия; использовать отраслевые категории
